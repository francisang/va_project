Visual Analytics Project: Singapore TV Market

Dimensional Charting to visualize the Singapore's Television market.

hide

Motivation to build this data visualization is to demonstrate interactivity between user and data. This visualization allows user to interact with data such as brushing, highlighting, or filtering in order to explore data. We make use history sales data Singapore's Television market data of to demonstrate interactivity data to find out in which area and what type of product or brand or TV features we should consider if we intend to open up a new TV store in Singapore.

Data will be illustrated in simple charts such as barchart and pie chart. The interactivity allows user to perform adhoc and oriented exploration.

DataFootPrints

The time series chart on the left represent total sales units from Jan 2010 to Sep 2013 for all brand of television sold in Singapore. It supports BrushOn function which allows user to specified time period of observation. Selecting specific in time series chart will set the filter for all the graphs. Selecting other component of horizontal barchart such as: 'Sales Unit by Brand' will allow data filtering by TV brand for that specific time period.

An example of oriented exploration is to find which area has highest sales units and what kind of television brand more popular and what features mostly people buy during specific period of time. Multiple filtering function allows users to observe particular parameter that suit their interest. The use of datatable will represent the details each item. Search typing and sort function in datatable give more interactivity for user by clicking the table header.
